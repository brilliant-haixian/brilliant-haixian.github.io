思科路由器操作密码相关

     console口密码设置
    
     命令如下（全部配置模式下）：
          Router(config)#line console 0
          Router(config-line)#password console
          Router(config-line)#login

     注意：
          console的密码是使用console线进行路由器配置时登录需要的密码

     password与secret密码设置

     命令如下（全部配置模式下配置，特权模式下保存设置）：
          Router(config)#enable password password
          Router(config)#enable secret secret
          Router(config)#exit
          Router#write


     注意：
          上面讲password的密码设置为password，将secret的密码设置为secret，其中secret的密码是默认加密的，
password的密码默认是不加密的，并且secret密码的优先级比 password密码的优先级高，即如果同时设置了
password和secret的密码，只有secret密码会生效。这两个密码是在用户模式切换到特权模式时需要的密码。

     vty密码设置（使用telnet连接时必须要设置的）

     命令如下
          Router(config)#line vty 0 15
          Router(config-line)#password vty
          Router(config-line)#login

     注意：
          这个密码是使用telnet进行远程管理时需要输入的密码。

     密码查看

     命令如下
          Router#show running-config

     密码加密

     命令如下
          Router(config)#service password-encryption
          Router(config)#exit
          Router#write

     将加密后的密码转为不加密的密码（默认加密的密码无法更改）
    
     命令如下
          Router(config)#no service password-encryption
          Router(config)#enable password password
          Router(config)#exit
          Router#write
     
     注意：
          no service password-encryption命令是无法将已经加密的密码解密的，因为加密是不可逆的，使用这个命令
后，接下来设置的密码就会明文显示。


     忘记密码重置密码操作

     操作如下
         
          我们忘记了进入特权模式的密码，那么我们就把路由器关机并重启！然后赶快按下【Ctrl+Break】键中断路
由器的启动过程，进入 rommon 模式！输入命令改变配置寄存器的值为 0x2142，这会使得路由器开机时不读取
NVRAM 中的配置文件 。然后重新启动路由器！
     命令如下：
rommon 1 > confreg 0x2142
rommon 3 > reset
     
     接下来操作如下：
          等到路由器重新启动后，我们会发现使用console口进入路由器和进入特权模式都不需要密码了，接下来我
们需要在特权模式下将配置文件加载到running-config文件中，并且进入全局配置模式修改console的登录密码以及
password和secret的密码，并且将寄存器的值恢复为正常值 0x2102，所以。
     命令如下：
          Router>enable
Router#copy startup-config running-config
Router#configure terminal
Router(config)#enable password password
Router(config)#enable secret secret
Router(config)#config-register 0x2102
Router(config)#exit
Router#write
Router#configure terminal
Router(config)#line console 0
Router(config-line)#password console
Router(config-line)#login
Router(config)#exit
Router#reload

               Router (config)#exit

               针对一些老式的路由器，我们还需要手动设定时钟频率，如果要设定时钟频率，我们必须要分清DTE与DCE的位置，因为是DCE（串行口母头）产生时钟频率，所以接下来我们使用命令查询哪一头是DCE头，并且设定时钟频率为56000。（具体的串口号结合自己的实际）

          命令如下：

               Router#configure terminal
               Router(config)#interface serial 2/0
               Router(config-if)#clock rate 56000

               Router (config)#exit

VLSM

(可变长子网掩码) 是为了有效的使用无类别域间路由（CIDR）和路由汇聚(route summary)来控制路由表的大小，网络管理员使用先进的IP寻址技术，VLSM就是其中的常用方式，可以对子网进行层次化编址，以便最有效的利用现有的地址空间。


Ospf 创建3个数据库：邻居表：show ip ospf neighbor；

拓扑表（LSDB）：show ip ospf database


路由表：show ip route



根据spf算法处理到达路由器的开销。创建 SPF 树。 然后使用 SPF 树计算最佳路由。 OSPF 将最佳路由放入转发数据库，用于创建路由表。

过程：
1建立邻居关系，
2 根据邻居的回话状态创建链路状态报告（LSA）且泛洪
3根据LSA创建LSDB（拓扑表）
4根据拓扑表执行spf算法计算最短路径 然后形成spf树
